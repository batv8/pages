Ghi nhớ Tailwind CSS qua Tư Duy Hệ Thống

Đừng cố gắng học thuộc lòng hàng ngàn classes. Phần mềm này giúp bạn nắm bắt Mental Model (Mô hình tư duy) cốt lõi của Tailwind, kết hợp thực hành tương tác (Interactive Practice) để tự động dịch Standard CSS sang Tailwind Classes trong đầu bạn.

1. Naming Convention (Quy tắc đặt tên)

Mọi class trong Tailwind đều tuân theo một công thức logic. Hiểu công thức này, bạn có thể tự suy luận ra 80% các class. Cấu trúc cơ bản thường là: [prefix]-[value] hoặc [property]-[scale].

Ví dụ: Spacing (Khoảng cách)

p - y - 4
  • p = padding (hoặc m = margin)
  • y = y-axis (Top & Bottom). Tương tự: x (Left/Right), t (Top), b (Bottom), l (Left), r (Right).
  • 4 = Scale (Thang đo). Mặc định 1 đơn vị = 0.25rem (4px). Vậy 4 = 1rem (16px).

Ví dụ: Typography (Chữ)

text - center
  • Bắt đầu với text- cho các thuộc tính liên quan đến chữ.
  • center = text-align: center;
  • Kích thước: text-lg, text-xl (Tương tự size quần áo: sm, md, lg, xl).
  • Màu sắc: text-red-500 (Property + Color + Shade).

2. CSS to Tailwind Dictionary

Bảng tra cứu nhanh các mapping phổ biến. Hãy chọn từng nhóm (Category) để xem cách chuyển đổi từ Standard CSS sang Tailwind.

flex-direction: column; flex-col
justify-content: space-between; justify-between
align-items: center; items-center

gap: 1rem; gap-4
margin: 1rem; m-4
padding-left: 2rem; padding-right: 2rem; px-8
width: 100%; w-full
height: 100vh; h-screen
max-width: 1024px; max-w-screen-lg
font-size: 1.5rem; line-height: 2rem; text-2xl
font-weight: 700; font-bold
text-align: right; text-right
color: #ffffff; text-white
border-radius: 0.5rem; rounded-lg
border-width: 1px; border
border-color: #e5e7eb; border-gray-200
box-shadow: ... shadow-md
